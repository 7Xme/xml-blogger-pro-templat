ID Card Processor

Upload front and back images of the ID card, manually crop them, apply brightness enhancement, and generate a PDF.

Front Side

1. Upload image. 2. Click Start Cropping. 3. Adjust crop area. 4. Click Crop.

Back Side

1. Upload image. 2. Click Start Cropping. 3. Adjust crop area. 4. Click Crop.

Note: Crop both front and back images for best results.
