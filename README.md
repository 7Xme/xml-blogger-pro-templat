## üì¶ Installation & Setup

### For Development
1. **Clone the repository**:
   ```bash
   git clone https://github.com/your-username/web-tools-collection.git
   cd web-tools-collection
   ```

2. **Set up a local server**:
   - Use any static file server (Apache, Nginx, etc.)
   - Or use Python's built-in server:
     ```bash
     python -m http.server 8000
     ```
   - Or use Node.js http-server:
     ```bash
     npx http-server
     ```

3. **Open in browser**:
   - Navigate to `http://localhost:8000` (or your server port)

### For Blogger Deployment
1. **Upload template files**:
   - Go to your Blogger dashboard
   - Navigate to Theme > Edit HTML
   - Replace the existing template with the content from `blogger-templates/`

2. **Upload tool files**:
   - Upload all HTML files from the respective tool directories
   - Ensure correct file paths in your Blogger posts/pages
